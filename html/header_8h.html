void* create_struct_header ( heap_t *  h,
char *  format_string,
void *  heap_ptr 
)

Creates a header and saves it on the heap.

This function is called when the header is to be created for data represented by a format string. If there are pointers in the format string the header created will be of the type STRUCT_REP, else it will be converted into RAW_DATA.

The format_string will be copied and doesn't need to be placed on the heap before hand.

Parameters
hthe heap to allocate format_string in if it is too big
format_stringthe string representation of the structure to create a header for
heap_ptrthe place on the heap where the header will be saved
Returns
pointer to where the data should be placed
