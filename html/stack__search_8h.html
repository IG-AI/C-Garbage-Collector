Finds a possible pointer on the stack.

Parameters
stack_bottomthe bottom of the stack to seach
stack_topthe top of the stack to seach
heap_startthe start of the heap
heap_endthe last position in the heap to which a pointer is allowed to refer
Returns
a pointer to a location in the stack between stack_bottom and stack_top that contains a possible address into the heap between heap_start and heap_end, or NULL if search is finished

Before calling stack search registers need to be dumped to the stack. This can be achieved by: #include <setjmp.h> #define Dump_registers() \ jmp_buf env; \ if (setjmp(env)) abort(); \

When calling the function use; __builtin_frame_address(n); To get a pointer to the start of a scopes memory on stack. <n> specifies what scope. 0 is the current, 1 the one above. On SPARC with Sun C Compiler this doesn't work so another method is needed.

extern char **environ; To get the bottom of the programs' stack. When refering to it use something similar to: void *stack_bottom = (void *)*environ; void *pointer = stack_find_next_ptr(&stack_bottom, stack_top, heap_start, heap_end);
